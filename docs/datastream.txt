Data stream
===========

Many datastreams exist, to be able to connect different modules between them.

List of data stream type
------------------------

* blob: generic data container, used for blob/fids/... (check container.txt)
* trackedblob: generic data container, used for blob/fids/..., and tracked
* mAST: multimodal abstract syntax tree, used between many Fusion modules as a generalized tree in the app's grammar
* audio: Recorded audio (in system's bitrate, etc.)
* JSON: String formatted JSON data
